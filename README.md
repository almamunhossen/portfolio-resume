# Portfolio-Resume Website

## Description

This is a personal portfolio and resume website for Al Mamun Hossen, a web developer. The website showcases skills, services, projects, education, work experience, testimonials, and contact information in an interactive and responsive design.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Dark/Light Mode Toggle**: Switch between themes for better user experience.
- **Animated Sections**: Smooth animations using AOS (Animate On Scroll) library.
- **Interactive Sliders**: Testimonials and projects displayed using Swiper.js.
- **Social Media Integration**: Links to Facebook, Behance, Twitter, GitHub, and YouTube.
- **Contact Form**: Easy way for visitors to get in touch.

## Technologies Used

- **HTML5**: Structure of the website.
- **CSS3**: Styling and layout, including custom gradients and responsive design.
- **JavaScript**: Interactive functionality, including theme toggle and navigation.
- **Font Awesome**: Icons for menu items and social links.
- **Swiper.js**: For creating touch-friendly sliders.
- **AOS (Animate On Scroll)**: For scroll-triggered animations.

## Installation and Setup

1. **Clone the Repository**:
   ```
   git clone https://github.com/almamunhossen/portfolio-resume.git
   cd portfolio-resume
   ```

2. **Open in Browser**:
   - Simply open `index.html` in your preferred web browser.
   - No server setup required as it's a static website.

## Usage

- **Navigation**: Use the menu icons to jump to different sections (About, Skills, Services, etc.).
- **Theme Toggle**: Click the sun/moon icon in the header to switch between light and dark modes.
- **Contact**: Fill out the contact form or use the provided social media links.
